QuizVerse BD

Project information

  • Category: Game App's
  • Project date: 12 Journey, 2025

This is a quiz app that includes various features. It has been specifically designed for the people of Bangladesh. The app's benefits include the ability to learn a lot of unknown facts from quizzes. Additionally, users can earn income through accurate answers. There is also a ranking system where users can see their rank based on their performance.

  • Multiple Quiz.
  • Admin App Controller
  • LeaderBoard System
  • Payment System
  • Multiple Theme.
  • Responsive Ui Design
  • Animation's
  • Push Notifications System
